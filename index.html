HTML

  • HTML: Hyper Text Markup Language
  • This language contains the foundational elements of building a website
  • Head elements include the meta data, which is super important for SEO

CSS

  • CSS: Cascading Style Sheets
  • This language builds on HTML to create style and pizazz of a website
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • There are three types of CSS Styles: Inline, Internal Style, and External
  • CSS rules consist of a property: value; which make up a declaration
  • Common CSS properties include: color, height, width, background-color, margin, border, box-shadow
  • CSS selectors include: Universal, id, and class

Git

  • Git is a distributed version control system.
  • Git allows developers the capability to make changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development
  • git clone repository url: Downloads the latest version of a remote project and puts it on the local machine
  • git clone repositoryURL -b brandName: Clones a specific branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript allows for sites to be functional & interactive
  • Array: Is a group of related data held together by a single array. It can be different types of data, and strings need to be contained in quotation marks
  • For Loops: Use the predictable pattern of indices to performa task on all the items in an array by allowing a single block of code t o be executed over and over
  • Functions: A set of instructions that tells the computer how to perform a certain task. These are different from for loops or conditional statements because functions do not automatically executed when the javascript file is run
